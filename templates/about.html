How to Add Pages

Here's a step-by-step guide to adding new pages to your SSR website:

Step 1: Add a Route

In main.py, add a new route handler:

@app.get("/your-page", response_class=HTMLResponse)
async def your_page(request: Request):
    return templates.TemplateResponse(
        "your_page.html",
        {
            "request": request,
            "title": "Your Page Title",
            "data": "Your data here"
        }
    )

Step 2: Create a Template

Create a new file in templates/your_page.html:

{% extends "base.html" %}

{% block content %}
<h1>{{ title }}</h1>
<p>Your content here</p>
{% endblock %}

Step 3: Update Navigation (Optional)

Add a link in templates/base.html navigation section.
