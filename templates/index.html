Getting Started

To add a new page, follow these steps:

  1. Create a new route in main.py
  2. Create a corresponding template in templates/
  3. Add a link to your navigation in templates/base.html

Database Examples

This template includes SQLAlchemy ORM examples:

  • View Users - See all users from database
  • View Posts - See all published posts
  • Create Example Data - Add sample data to database

Check models.py for database model definitions and main.py for ORM usage examples.
