ORM Code Example

This page uses the following ORM query:

posts = db.query(Post)
    .filter(Post.is_published == True)
    .order_by(Post.created_at.desc())
    .all()

This demonstrates:

  • .filter() - Filter by published status
  • .order_by() - Sort results by creation date (newest first)
  • .all() - Execute and return all results
