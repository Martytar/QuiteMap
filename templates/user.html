User Profile: {{ username }}

This is an example of a dynamic route using path parameters.

The username {{ username }} was extracted from the URL path.

How Dynamic Routes Work

In main.py, the route is defined as:

@app.get("/user/{username}", response_class=HTMLResponse)
async def user_profile(request: Request, username: str):
    return templates.TemplateResponse(
        "user.html",
        {
            "request": request,
            "title": f"User: {username}",
            "username": username
        }
    )

The {username} in the path becomes a parameter that you can use in your template.
